### Attendance Management System:

# User Types:

- Host
- Participant

# Features:

- Authentication (with Firebase):

  - Register (Email & Password) [Status: Done]
  - Login (Email & Password) [Status: Done]

- Event Management:

  - Create Event
  - Read All Event
  - Read Single Event
  - Update Event
  - Delete Event
  - Join Event

- Session Management:
  - Create Session
  - Read All Session
  - Read Single Session
  - Update Session
  - Delete Session
  - Join Session

- Notification:
  - Remind Sessions
  - Alert Message to Host

- Messaging:
  - One Direct Message to Host

- Approval:
  - Approve Permission
